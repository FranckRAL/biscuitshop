## 📖 About

BiscuitShop est une plateforme e-commerce moderne permettant aux clients de découvrir, commander et payer leurs biscuits préférés en quelques clics.

L'application met l'accent sur une expérience utilisateur fluide, un design chaleureux inspiré des couleurs du beurre et des biscuits, ainsi qu'un système de paiement sécurisé grâce à l'intégration de l'API MVola.

## ⚙️ Installation

git clone https://github.com/username/BiscuitShop.git

cd BiscuitShop

python -m venv .venv

source .venv/bin/activate

pip install -r requirements.txt
